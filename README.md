# 👑 Dynasty

> A premium, high-performance Music PWA designed for the modern era.

![banner](https://images.unsplash.com/photo-1470225620780-dba8ba36b745?auto=format&fit=crop&w=1200&q=80)

## ✨ Overview

Dynasty is a state-of-the-art music application that combines sleek design with powerful functionality. Built with modern web technologies, it offers a seamless and immersive listening experience.

## 🚀 Key Features

- **Blazing Fast**: Built with Vite and React for near-instant load times.
- **Glassmorphic UI**: Beautifully crafted interface with modern aesthetics.
- **Dynamic Content**: Powerful search and discovery features.
- **PWA Ready**: Install it on your device for a native-like experience.
- **TypeScript Core**: Robust, type-safe implementation for reliability.

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Node.js, Fastify, Prisma
- **Styling**: Tailwind CSS 4
- **Database**: SQLite (file-based, zero infrastructure)

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kotelyanets/Dynasty.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment:
   - Copy `.env.example` to `.env`
   - Configure your `MUSIC_DIR` and `VITE_API_URL`

4. Run the development server:
   ```bash
   npm run dev
   ```
